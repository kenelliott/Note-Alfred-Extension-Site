Notes Management with Alfred


Notes is a simple extension for Alfred that lets you easily manage entries in Mountain Lion's Notes app. Alfred is a top notch productivity app for Mac OS X that removes the mouse from common tasks. If you haven't been introduced, get over to http://www.alfredapp.com and check him out.

Installation

If you have the latest version of Alfred and the Powerpack, installation will be a breeze! Simply download the extension file by clicking on the download link and double click on the file or drag and drop it onto Alfred's preferences pane and it will be installed and ready to use. No need to restart Alfred, he's smart!

Usage

Open Alfred and enter your note entry in this format note title:string. Where 'note' is the command telling Alfred to use the Note extension. title is the name/title of the note and string equals the new entry. If a note with the a title is found, string will be appended to that note on a new line. If a note with that name does not exist, it will be created and string will be added as the first entry. Currently, Notes is a new extension and requires the exact entry format.

Examples

  • note work:create a way to add notes
  • note work:updating notes would be nice also
  • note play:pick up more silly string
  • note shopping:more Nutella!
